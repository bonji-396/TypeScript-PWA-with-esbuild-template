# TypeScript + PWA のテンプレート

esbuildバンドラーを用いて、TypeScriptでPWAのフロントエンドアプリを作るためのテンプレートです。



## 主な技術構成
- TypeScript
- PWA
- Sass
- RxJS

## 導入

以下を実行
```zsh
npm install
```
